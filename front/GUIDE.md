## AlphaNote란?

AlphaNote는 **성장을 기록하고 공유하는 플랫폼**입니다.

매일의 습관을 추적하고, 학습한 내용을 기록하며, 커뮤니티와 소통하면서 꾸준한 성장을 이어갈 수 있도록 돕습니다.

### 핵심 가치
- **기록**: 습관, 학습, 목표를 체계적으로 기록
- **시각화**: GitHub 스타일 활동판으로 한눈에 확인
- **공유**: 커뮤니티와 경험을 나누며 함께 성장

---

## 시작하기

### 1. 회원가입 / 로그인
1. 홈페이지 우측 상단의 **로그인** 버튼 클릭
2. 이메일 회원가입 또는 Google 소셜 로그인 선택
3. 회원가입 후 자동으로 로그인됩니다

### 2. 첫 화면 둘러보기
- **최근 질문**: 커뮤니티의 최신 Q&A 확인
- **최근 습관**: 등록된 습관 트래킹 현황
- **인기 성장기록**: 조회수 기준 인기 글 모음

## 주요 기능

### 1. 습관 트래킹

#### 습관 만들기
1. 상단 메뉴에서 **Habit** 클릭
2. 우측 상단 **새 습관 만들기** 버튼 클릭
3. 습관 정보 입력:
   - **습관명**: 추적할 습관의 이름 (예: "물 2L 마시기")
   - **목표 횟수**: 하루 목표 달성 횟수 (예: 8회)
   - **색상**: 캘린더에 표시될 색상 선택
   - **시작일**: 습관 추적 시작 날짜
   - **종료일** (선택): 기간 제한이 있는 습관인 경우 설정
   - **설명** (선택): 습관에 대한 추가 설명

#### 습관 기록하기
1. 습관 목록에서 원하는 습관 카드 클릭
2. 상단의 **+** 버튼으로 오늘의 기록 추가
3. 횟수 입력 또는 메모 작성
4. **기록하기** 버튼으로 저장

#### 습관 캘린더 보기
- **6개월 활동판**: GitHub 스타일의 시각화로 한눈에 진행상황 확인
- **색상 강도**: 목표 달성률에 따라 색상이 진해집니다
  - 회색: 기록 없음
  - 연한 초록: 50% 미만
  - 초록: 50~100%
  - 진한 초록: 100% 달성
  - 가장 진한 초록: 목표 초과

#### 통계 확인
- **현재 연속**: 연속으로 기록한 일수
- **최장 연속**: 역대 최장 연속 기록
- **총 기록**: 전체 기록 횟수
- **일일 목표**: 설정한 하루 목표 횟수

### 2. QnA 커뮤니티

#### 질문하기
1. 상단 메뉴에서 **QnA** 클릭
2. 우측 상단 **질문하기** 버튼 클릭
3. 질문 정보 입력:
   - **제목**: 질문의 핵심을 담은 제목 (최소 5자)
   - **카테고리**: 기술 / 커리어 / 기타 중 선택
   - **내용**: 자세한 질문 내용 작성 (최소 10자)
   - **태그** (선택): 관련 키워드 추가 (엔터로 구분)

#### 답변하기
1. 질문 카드를 클릭하여 상세 페이지로 이동
2. 하단 **답변하기** 영역에 답변 작성
3. **답변 등록** 버튼으로 제출

#### 답변 채택하기 (질문 작성자만 가능)
1. 자신의 질문에 달린 답변 중 채택할 답변 선택
2. **채택하기** 버튼 클릭
3. 채택된 답변은 초록색 체크 아이콘으로 표시됩니다

#### 필터 및 검색
- **카테고리 탭**: 전체 / 기술 / 커리어 / 기타
- **정렬 옵션**: 최신순 / 답변순 / 조회순
- **검색**: 제목, 내용, 작성자로 검색 가능
- **내 질문**: 로그인 시 자신이 작성한 질문만 필터링

#### 질문 상태
- **답변 배지**: 왼쪽 상단에 답변 개수 표시
- **해결 표시**: 답변이 채택되면 "해결" 배지로 변경 + 체크 아이콘

### 3. 성장기록

#### 글 작성하기
1. 상단 메뉴에서 **Growth** 클릭
2. 우측 상단 **글 작성** 버튼 클릭
3. 글 정보 입력:
   - **제목**: 기록의 제목
   - **카테고리** (선택): 분류 태그
   - **내용**: Markdown 형식으로 작성 가능
   - **썸네일** (선택): 대표 이미지 업로드
   - **공개 설정**: 공개 / 비공개 선택
4. **임시저장** 또는 **발행하기**

#### 글 관리
- **전체 글 보기**: 커뮤니티의 모든 공개 글
- **내 글**: 내가 작성한 글 (공개/비공개/임시저장 포함)
- **글 수정/삭제**: 내 글 상세 페이지에서 가능

#### 검색 및 필터
- **정렬**: 최신순 / 인기순 / 댓글순
- **검색**: 제목, 내용, 작성자로 검색

#### 임시저장 기능
- 작성 중인 글을 임시저장하여 나중에 이어서 작성 가능
- 임시저장 글은 본인만 볼 수 있습니다

### 4. 목표설정

#### 연간 목표 설정
1. 상단 메뉴에서 **Goal** 클릭
2. 연도를 선택하고 **새 목표 추가** 버튼 클릭
3. 목표 정보 입력:
   - **목표 제목**: 달성하고자 하는 목표
   - **목표 설명**: 상세한 내용
   - **공개 설정**: 다른 사람에게 공개할지 선택

#### 목표 달성 체크
- 목표 카드의 체크박스를 클릭하여 달성/미달성 토글
- 달성한 목표는 시각적으로 구분됩니다

#### 다른 사람의 목표 보기
- **공개 목표 탭**에서 커뮤니티의 목표들을 확인
- 서로의 목표를 보며 동기부여 받기

### 5. AI 멘토링 (개발 중)

> 이 기능은 현재 개발 중이며, 곧 출시될 예정입니다.

#### 개요
AI 멘토링은 **내가 작성한 성장기록을 기반으로 AI와 대화하며 학습을 확장**하는 기능입니다.

#### 주요 특징
- **맥락 기반 학습**: 내가 작성한 성장기록을 AI가 분석하여 맥락에 맞는 질문과 답변 제공
- **능동적 학습**: 단순 검색이 아닌, 기록한 내용을 토대로 더 깊이 있는 질문을 주고받을 수 있습니다
- **학습 연결고리**: 기록이 곧 학습으로 이어지는 새로운 성장 경험

#### 활용 시나리오
1. 성장기록에 학습한 내용을 작성
2. AI 멘토링에서 해당 주제에 대해 추가 질문
3. AI가 내 기록을 참고하여 맞춤형 답변 제공
4. 대화 내용을 다시 성장기록으로 정리

#### 기대 효과
- **개인화된 학습 경험**: 나만의 학습 히스토리를 바탕으로 한 맞춤형 멘토링
- **지식 확장**: 단편적인 기록을 체계적인 학습으로 연결
- **학습 동기 유지**: AI와의 대화를 통해 지속적인 학습 동기 부여

### 6. 성장 대시보드 (개발 중)

> 이 기능은 현재 개발 중이며, 곧 출시될 예정입니다.

#### 개요
성장 대시보드는 **습관 기록, 성장기록, 목표 달성 이력을 하나로 모아 나만의 성장 페이지**를 만드는 기능입니다.

#### 주요 특징
- **통합 대시보드**: 습관 트래킹, 성장기록, 목표 등 모든 활동을 한눈에 확인
- **시각화**: 시간의 흐름에 따른 성장 과정을 그래프와 차트로 표현
- **개인 브랜딩**: 나만의 성장 스토리를 시각적으로 표현

#### 포함될 정보
- **활동 요약**: 습관 달성률, 기록 작성 현황, 목표 달성률
- **타임라인**: 시간 순으로 정리된 성장 기록
- **통계 분석**: 월별/연도별 활동 패턴 분석
- **주요 마일스톤**: 중요한 성취와 전환점 하이라이트

#### 활용 방안
1. **자기 성찰**: 내 성장 과정을 돌아보고 다음 목표 설정
2. **포트폴리오**: 개발자, 디자이너 등 전문가의 성장 과정 증명
3. **이력서 대체**: 단순 경력이 아닌, 실제 성장 과정을 보여주는 자료
4. **공유**: 공개 설정 시 다른 사람들에게 내 성장 과정 공유

#### 기대 효과
- **성장 가시화**: 추상적인 성장을 구체적인 데이터로 확인
- **동기 부여**: 쌓인 기록을 보며 성취감과 다음 목표에 대한 동기 부여
- **차별화**: 남들과 다른 나만의 성장 스토리 구축

## Tips & FAQ

### 💡 효과적인 사용 팁

#### 습관 트래킹
- **작은 목표부터**: 처음에는 달성 가능한 작은 목표로 시작하세요
- **매일 기록**: 가능한 한 매일 같은 시간에 기록하는 습관을 들이세요
- **연속 기록**: 연속 일수가 끊기지 않도록 노력하면 동기부여에 도움이 됩니다

#### 성장기록
- **Markdown 활용**: 코드 블록, 이미지, 링크 등을 활용해 풍부한 글 작성
- **꾸준한 기록**: 작은 배움이라도 꾸준히 기록하면 큰 자산이 됩니다
- **공개 vs 비공개**: 민감한 내용은 비공개로, 공유하고 싶은 내용은 공개로 설정

#### QnA 커뮤니티
- **구체적인 질문**: 문제 상황을 자세히 설명하면 더 좋은 답변을 받을 수 있습니다
- **태그 활용**: 관련 키워드를 태그로 추가하면 검색이 쉬워집니다
- **답변 채택**: 도움이 된 답변은 꼭 채택하여 감사를 표현하세요

### ❓ 자주 묻는 질문

**Q. 습관 기록을 수정하거나 삭제할 수 있나요?**
A. 네, 습관 상세 페이지에서 기록한 내용을 수정하거나 삭제할 수 있습니다.

**Q. 성장기록의 임시저장 글은 어디서 볼 수 있나요?**
A. Growth 페이지에서 "내 글" 탭 선택 후, "임시저장" 필터를 선택하면 확인할 수 있습니다.

**Q. 질문에 답변이 달렸을 때 알림을 받을 수 있나요?**
A. 우측 상단의 알림 아이콘을 통해 새로운 답변, 댓글 등의 알림을 확인할 수 있습니다.

**Q. 프로필 정보는 어디서 수정하나요?**
A. 우측 상단의 프로필 아이콘 클릭 → "프로필" 메뉴에서 수정 가능합니다.

**Q. 습관의 종료일을 설정하지 않으면 어떻게 되나요?**
A. 종료일이 없으면 계속해서 추적 가능한 습관으로 유지됩니다. 언제든 보관하거나 삭제할 수 있습니다.

**Q. 답변을 채택한 후에 다른 답변으로 변경할 수 있나요?**
A. 네, 가능합니다. 다른 답변의 "채택하기" 버튼을 클릭하면 기존 채택이 해제되고 새로운 답변이 채택됩니다.

**Q. 성장기록의 공개 설정을 나중에 변경할 수 있나요?**
A. 네, 글 상세 페이지에서 언제든지 공개/비공개를 전환할 수 있습니다.
